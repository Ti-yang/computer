# 整数安全计算器兼二进制计算 项目文档

## 项目概述
这是一个基于C语言开发的整数安全计算器程序，提供了基本的算术运算功能，并特别设计了防止整数溢出的安全机制。程序具有交互式命令行界面，支持二进制数字显示功能。

## 文件结构

### 1. over.h & over.c - 安全运算库

**over.h** - 头文件声明
- 包含必要的标准库头文件
- 声明四个安全运算函数

**over.c** - 安全运算实现
- `safe_add()`: 安全加法，检测正负溢出
- `safe_sub()`: 安全减法，检测边界情况  
- `safe_mul()`: 安全乘法，处理各种符号组合
- `safe_div()`: 安全除法，防止除零和特定溢出

### 2. func.h & func.c - 功能模块

**func.h** - 功能函数声明
- 包含项目所需的所有头文件
- 声明界面显示和运算功能函数

**func.c** - 功能函数实现
- **界面函数**:
  - `print_hello()`: 欢迎界面动画
  - `print_main()`: 逐字符打印效果
  - `print_choice()`: 操作选项菜单
- **运算函数**:
  - `add()`, `sub()`, `mul()`, `siv()`: 四则运算包装函数
  - `result()`：运算路由控制器
- **工具函数**:
  - `print_bin()`: 整数转二进制显示（支持负数补码）

### 3. main.c - 主程序

- 程序入口点
- 实现主循环和用户交互逻辑
- 集成所有功能模块

## 核心特性

### 安全运算机制
- **加法溢出检测**: 检查`a + b`是否超出`INT_MAX`或`INT_MIN`
- **减法溢出检测**: 处理边界特殊情况
- **乘法溢出检测**: 分情况处理不同符号组合
- **除法安全**: 防止除零和`INT_MIN / -1`溢出

### 用户界面特性
- 逐字符打印动画效果
- 菜单驱动的交互方式
- 清屏和暂停控制
- 二进制数字可视化显示

## 技术特点

1. **跨平台兼容**: 使用标准C库函数
2. **错误处理**: 完善的溢出和错误检测
3. **代码模块化**: 清晰的层次结构分离
4. **用户体验**: 友好的交互界面和视觉效果

## 编译运行
该项目采用标准C语法，可在支持C99标准的编译环境中直接编译运行。主要依赖Windows API的`Sleep()`函数，如需跨平台可相应调整。

这个项目展示了C语言在系统编程、安全计算和用户界面设计方面的综合应用能力。
